# 🏺 Guida Installazione Aura Bridge (Alpha v0.1.0)

L'estensione **Aura Bridge** ti permette di inviare contenuti web direttamente al tuo NeuralVault con un click. Ecco come installarla nella tua istanza locale.

### 📁 Posizione File
I file dell'estensione si trovano nella cartella:
`aura_extension/` all'interno della tua repository.

### 🛠️ Passaggi per l'installazione (Chrome / Brave / Edge)

1.  **Apri la gestione estensioni**:
    *   Digita `chrome://extensions/` nella barra degli indirizzi.
2.  **Attiva la "Modalità sviluppatore"**:
    *   Trova il toggle in alto a destra e attivalo.
3.  **Carica l'estensione**:
    *   Clicca sul pulsante **"Carica estensione non pacchettizzata"** (Load unpacked).
    *   Seleziona la cartella `aura_extension` che ho appena creato nella tua repository.
4.  **Fissa l'estensione**:
    *   Clicca sull'icona del puzzle nel browser e "pinna" Aura Bridge per averla sempre visibile.

### 🚀 Come usarla
1.  Assicurati che il backend di NeuralVault sia attivo (`python api.py`).
2.  Naviga su qualsiasi pagina web (es. documentazione tecnica).
3.  Clicca sull'icona di **Aura Bridge**.
4.  Clicca su **"Invia alla Nebula"**.
5.  Il contenuto della pagina verrà istantaneamente digerito dal tuo Vault e apparirà nella Nebula 3D.

> [!TIP]
> L'estensione cattura automaticamente il titolo della pagina e l'URL come metadati, permettendoti di ritrovare la fonte originale nella Nebula.
